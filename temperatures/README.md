# API de Temperaturas de Ciudades

API RESTful para gestionar y consultar temperaturas de diferentes ciudades construida con Django REST Framework.

## Características

- ✅ Operaciones CRUD completas (Create, Read, Update, Delete)
- ✅ Autenticación por sesiones
- ✅ Permisos: lectura pública, escritura para usuarios autenticados
- ✅ Panel de administración Django
- ✅ API documentada automáticamente
